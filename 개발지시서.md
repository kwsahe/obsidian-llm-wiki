## STEP 4. 자동화 스크립트

> `_system/scripts/daily_note_gen.py` 생성.
> Claude Code에서 직접 실행하거나 매일 아침 수동 실행용.

### 기능
- 오늘 날짜로 `02-calendar/YYYY-MM-DD.md` 자동 생성
- 이미 파일이 있으면 스킵
- `daily-note.md` 템플릿 구조 그대로 적용 (Templater 없이 Python으로 처리)

### 스크립트 구조
```python
import os
from datetime import datetime

VAULT_PATH = r"C:\Users\sangh\Desktop\Code\Obsidian\obsidian-llm-wiki"
CALENDAR_PATH = os.path.join(VAULT_PATH, "02-calendar")

def create_daily_note():
    today = datetime.now()
    date_str = today.strftime("%Y-%m-%d")
    day_str = today.strftime("%A")
    filename = f"{date_str}.md"
    filepath = os.path.join(CALENDAR_PATH, filename)

    if os.path.exists(filepath):
        print(f"이미 존재: {filename}")
        return

    content = f"""---
created: {date_str}
day: {day_str}
tags: [daily]
mood: 
focus: 
---

# {date_str} ({day_str})

## 오늘 할 일
- [ ] 
- [ ] 
- [ ] 

## 공부/개발 기록

## 에러 & 해결

## 취준 활동

## 내일 할 일
- [ ] 
"""
    with open(filepath, "w", encoding="utf-8") as f:
        f.write(content)
    print(f"생성 완료: {filename}")

if __name__ == "__main__":
    create_daily_note()
```
