# O2O 수요 예측 서비스 (O2O-Demand-Forecasting-Solution)

## 개요
국토교통부 아파트 매매·전월세 실거래가, 한국부동산원 입주예정물량, 건축HUB 대수선 이력 데이터를 결합해 시군구별(서울·경기·인천·5대 광역시 총 102개) **인테리어 수요 점수(0~100점)** 를 산출하는 분석 파이프라인. 공공데이터 API 자동 수집 → 분석 → Flask 대시보드(랭킹/차트/네이버 지도) → LLM 챗봇까지 end-to-end로 동작하며, 첫 집 구매자를 위한 "구매 가이드" 탭도 제공한다.

## 기술 스택
- **수집**: 공공데이터포털 API (국토교통부 실거래가, 건축HUB 대수선 이력, 전국인테리어업체표준데이터, 소상공인 상가정보), 행정안전부 인구현황 CSV
- **파이프라인**: Python, pandas (`src/pipeline.py` — 7개 지표 Min-Max 정규화 + 가중합산)
- **시계열 예측**: SARIMA, Prophet, LightGBM (월별 거래량 예측 비교)
- **백엔드**: Flask (`app.py`) — 대시보드 + REST API + 챗봇 2종
- **AI 챗봇**: 로컬 Ollama (qwen2.5:3b, GPU 구동), 질문/응답 SQLite(`chat_history.db`) 기록
- **지도**: Naver Maps API (Naver Cloud Platform)
- **시각화**: 대시보드(HTML/JS), Jupyter 노트북 4종

## 주요 기능
- **인테리어 수요 점수 산출**: 거래건수(20%)·전월세거래건수(20%)·거래금액(15%)·노후도(15%)·전용면적(10%)·신규입주(10%)·대수선이력건수(10%) 7개 지표를 Min-Max 정규화 후 가중합산, S(60~100)/A(30~59)/B(0~29) 등급 + `+`/`0`/`-` 세분화
- **핵심 타겟 세그먼트**: 노후도 15~20년(Old_Apartment) 아파트를 전면 리모델링 핵심 타겟으로 정의
- **인터랙티브 대시보드**: KPI 카드, TOP 10 바 차트, 세그먼트 도넛 차트, 랭킹 테이블, 네이버 지도(수요등급/시장규모·시공비 2가지 모드 토글), 시도 필터, 지역명·단지명 검색
- **AI 챗봇 2종**: 수요 점수 데이터 기반 질의응답(오타·부분일치 매칭 지원) + 첫 집 구매 가이드 전용 챗봇
- **수요 예측 탭**: 월별 거래량을 SARIMA/Prophet/LightGBM으로 예측해 MAPE 기준 비교
- **정보성 컬럼**: 인테리어업체수, 예상시공비(하한/상한), 시장규모 추정(억 원), 총인구수/청년·고령인구비율 — 수요 점수 산출에는 미포함, 대시보드 참고용
- **API 자동 수집**: `/api/collect`로 최근 N개월 실거래가 재수집 + 파이프라인 재실행
